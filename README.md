## ✨ 新增与优化功能

* **解除修改限制**：适配平台部分防守地图的超大属性（支持兆、京、垓、秭、穰、沟、极等单位）。默认上限值为 **7**（百万单位）。
* **快捷键操作**：新增全局快捷键，按下 `F5` 快速刷新数据，按下 `F6` 一键执行修改。
* **窗口总在最前**：程序界面默认置顶，完美适配《魔兽争霸 III》窗口化运行模式。

## ⚠️ 注意事项

> 本工具仅限单机娱乐测试使用，**不支持多人联机**。
