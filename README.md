## 📝 简介
**War3Trainer** 是一款专为《魔兽争霸 III》(Warcraft III) 设计的内存修改工具。本项目通过直接读取和分析游戏进程的内存数据，精确定位游戏中的特定地址，从而实现对游戏内英雄属性等数据的实时修改。

---

## 🚫 注意事项与限制
本项目**不属于**常规意义上的网络作弊器，使用前请务必了解以下限制：
1. **仅限单机**：本修改器只能在单机模式（自定义游戏/战役）下使用。
2. **无法用于对战**：**不适用**于任何形式的网络对战（如官方战网或对战平台），更无法在战网上生效。本程序在设计上**没有**向网络发送过任何欺骗性信息。
3. **局域网联机机制**：如果你想在局域网内和朋友联机打 RPG 地图，**需要在每台参与联机的电脑上进行完全相同的修改动作**。如果不这样做，游戏数据不同步会立刻导致掉线。只有同步修改才能保障不掉线。
